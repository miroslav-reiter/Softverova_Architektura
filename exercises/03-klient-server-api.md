# 03 - Klient-server model a API

Cieľ: pochopiť tok požiadavky medzi klientom, serverom a databázou.

## Zadanie

Navrhni jednoduché API pre produktový katalóg e-shopu.

Navrhni endpointy:

1. zoznam produktov,
2. detail produktu,
3. vytvorenie produktu,
4. aktualizácia produktu,
5. odstránenie produktu.

## Výstup

Pre každý endpoint uveď:

- HTTP metódu,
- URL cestu,
- stručný popis,
- príklad odpovede.

## Pomôcka

Použi GET, POST, PUT a DELETE.
