# 06 - Kvalitatívne vlastnosti architektúry

Cieľ: pochopiť, že architektúra je o kompromisoch medzi vlastnosťami systému.

## Zadanie

Pre e-shop urč priority týchto vlastností:

1. Performance
2. Scalability
3. Availability
4. Security
5. Reliability
6. Maintainability

## Úloha

Pre každú vlastnosť uveď:

- prečo je dôležitá,
- ako ju architektúra ovplyvňuje,
- aký kompromis môže vzniknúť.

## Výstup

Vytvor tabuľku s prioritou od 1 do 5 a krátkym zdôvodnením.
