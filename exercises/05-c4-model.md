# 05 - C4 model

Cieľ: naučiť sa opísať systém cez C4 model.

## Zadanie

Pre e-shop vytvor textový návrh C4 modelu.

Popíš:

1. Context - kto systém používa a s čím komunikuje.
2. Container - z akých aplikácií sa skladá.
3. Component - aké moduly obsahuje backend.
4. Code - aké triedy alebo funkcie by mohli byť v order module.

## Výstup

Vytvor štyri sekcie: Context, Container, Component, Code.

## Pomôcka

Na začiatku stačí textový model. Diagram môžeš neskôr vytvoriť v Mermaid alebo PlantUML.
