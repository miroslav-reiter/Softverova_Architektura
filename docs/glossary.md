# Slovník pojmov - Softvérová architektúra

## Architektúra

Základné koncepty, vlastnosti, prvky, vzťahy a princípy návrhu a vývoja systému.

## Komponent

Samostatná časť systému s jasnou zodpovednosťou.

## Monolit

Architektúra, kde je systém vytvorený a nasadzovaný ako jeden celok.

## Mikroservis

Samostatná služba s jasnou zodpovednosťou, ktorá komunikuje s ostatnými službami cez definované rozhrania.

## API

Application Programming Interface. Kontrakt medzi systémami alebo komponentmi.

## REST

Architektonický štýl pre návrh webových API založený na resource modely a HTTP metódach.

## C4 model

Model vizualizácie architektúry cez úrovne Context, Container, Component a Code.

## UML

Unified Modeling Language. Jazyk na modelovanie a vizualizáciu systémov.

## Technical Debt

Technický dlh. Cena za rýchle alebo zlé technické a architektonické rozhodnutia.

## Quality Attributes

Kvalitatívne vlastnosti systému, napríklad výkon, škálovateľnosť, bezpečnosť a udržiavateľnosť.

## ADR

Architecture Decision Record. Dokument zachytávajúci architektonické rozhodnutie, jeho kontext a dôsledky.
